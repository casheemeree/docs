1. Отправьте боту команду <kbd>/start</kbd> и нажмите <kbd>Добавить устройство</kbd>
2. Откроется страница настройки VPN
3. Установите приложение Happ
4. После установки приложения, вернитесь на страницу настройки.

   <Frame>
     ![Image 11](/images/image-11.png)
   </Frame>
5. Нажмите <kbd>Активировать VPN</kbd>, чтобы импортировать ключ-доступа в приложение​
